Odoo Website Order Cancel App

This apps helps website user to cancel website order from my account page of website. User can easily login to their portal account and using my account option they can view all place orders and easily cancel their eCommerce-webshop order from website.

Request to cancel the order.

User can request to cancel the order from the list view of orders.

Reasons and Remarks.

User can give the reason the cancel the order and also can give the remarks for order.

Approval to Cancel the Order.

Administrator can approve the request to to cancel the order.
